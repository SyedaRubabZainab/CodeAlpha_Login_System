# 🔐 Login & Registration System (C++)

## 📌 Overview

A secure login and registration system using file handling and password hashing.

---

## ⚙️ Features

* User registration
* Secure login
* Password hashing
* Input validation
* Duplicate username check

---

## ▶️ How to Run

### 🔹 Run using EXE

Double-click:
login_system.exe

### 🔹 Run using Compiler

g++ login_registration.cpp -o login
./login
